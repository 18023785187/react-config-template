# React项目配置模板

```javascript
// 配合hym-react-cli使用
$ npm i -g hym-react-cli
$ npx hym-react-cli <project_name> --template <template>
// 详细说明看 https://github.com/18023785187/hym-react-cli.git
```

# 启动本地服务

```javascript
$ git clone https://github.com/18023785187/react-config-template.git
$ npm install
$ npm run dll
$ npm run start
```
